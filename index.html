Документация API

Для использования API отправьте POST запрос на:

/api/translate

Параметры запроса:

  • sourceLang: исходный язык ('ru' или 'kchr')
  • targetLang: целевой язык ('ru' или 'kchr')
  • text: текст для перевода
  • apiKey: ваш API ключ

Пример ответа:

{
    "success": true,
    "translatedText": "Переведенный текст",
    "sourceText": "Исходный текст"
}
